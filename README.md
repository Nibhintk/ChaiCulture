# Chai Culture – Coming Soon Landing Page

A premium “Coming Soon” landing page for a luxury Indian chai brand inspired by royal households.

## Features
- Elegant, royal-inspired design
- Mobile-responsive layout
- Email signup form
- Subtle animations with accessibility support
- Social media links

## Tech Stack
- HTML
- CSS
- Font Awesome

## Live Demo
https://chaiculture-coomingsoon.netlify.app/

## Design Note
The design uses warm cream tones with gold and brown accents to reflect Indian heritage and a premium, timeless feel. Typography and spacing are kept minimal to maintain elegance and focus on the brand.
